Connect to SAP Ariba Analytical

Navigate to the Ariba developer login and sign in. Click "Manage" (bottom icon on the left). Click the + button next to the search field. Set the "Application Name" and "Description" to Precog then click "Submit". Click "Actions" then click "Request API Access". Select the API of interest, then select your Realm and click "Submit". You may need to wait 12 hours for this request to be approved.

Once approved click "Actions" then click "Generate OAuth Secret". Copy the Application Key, OAuth Client Id and OAuth Secret and paste these credentials in the configuration form below.
